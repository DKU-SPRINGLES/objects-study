## 📚 진행 방법

매일(월~금)마다 `2개 장`의 내용을 정리한 것을 공유 한다.

정리해야할 내용 :

- (필수) 간략한 정리
- (필수) 읽고 느낀 점
- (선택) 궁금한것 혹은 서로 이야기하고 싶은 주제

궁금한것, 이야기하고 싶은 주제는 이슈로 남기고 서로의 의견을 댓글로 남긴다.

## 📚 공유 방법

각 장, 절에 맞는 폴더에 들어가서 본인 이름의 마크다운을 만든 다음, 거기에 내용을 정리하기.

ex) 산하는 [sanha.md](http://sanha.md) 파일에 내용을 정리
